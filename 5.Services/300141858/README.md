# Laboratoire Services AD - 300141858

## service1
Commande :
`.\services1.ps1`

Liste les services Active Directory (NTDS, ADWS, DFSR, kdc, Netlogon, IsmServ) et affiche leur état.

## service2
Commande :
`.\services2.ps1`

Affiche les événements récents du service Directory Service et les logs Netlogon.

## service3
Commande :
`.\services3.ps1`

Exporte les événements Active Directory dans C:\Logs\ADLogs.csv.

## service4
Commande :
`.\services4.ps1`

Arrête le service DFSR, affiche son état, puis le redémarre.
